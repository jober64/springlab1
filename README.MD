# Fake balances service
##### Web service implementation example using:
* JAVA 8
* Springboot
* H2 database (embeded)
* Tomcat (embeded)
* RestTemplate
* ...

##### Exposed API
```
/balances
```
returns fake banalces for set of companies

##### Usage
Build project with Maven and start as plain JAVA application
```
mvn package
java -jar target/application.jar
```
